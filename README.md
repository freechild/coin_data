# coin_data

## python api server 

### info
> 1. lenguage : python
> 2. framework : django 
> 3. DB : mongodb (pymongo)
### purpose 
> 1. 특정 사이트 웹소켓 연결 후 일정 데이터와 타임스탬프 포함하여 db에 저장
> 2. 저장된 db데이터 json타입으로 전송
